Waggle
===

Waggle is a set of modules to transform Drupal into a powerful ticketing
and project management platform.

Waggle's architecture provides:
	* E-mail integration
	* Search-driven interfaces
	* Analytics
	* Group-guided estimation
	* Calendar and timeline
	* Hashtags
	* Global change tracker


waggle_email
------------

Waggle's email system requires a few extra PHP libraries:
	* Mail_Mbox (install from PEAR: sudo pear install channel://pear.php.net/mail_mbox-0.6.3)
	* Mail_MimeDecode (install from apt-get or PEAR: sudo apt-get install php-mail-mimedecode)
